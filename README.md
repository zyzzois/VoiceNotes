# VoiceNotes

### Мною дополнительно реализовано:
* Распознование голосовой заметки в текстовый формат by https://speechtext.ai/
* Авторизация через VK SDK
* Сохранение аудио-заметок в документы VK


### Стек технологий
* Kotlin
* MVVM
* Dagger 2
* Coroutines
* Room
* VK SDK
* Recycler, List Adapter, DiffUtils
* View Binding
